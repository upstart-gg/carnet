# Greeting Skills

## Overview
This skill provides capabilities for greeting users and introducing yourself in a friendly manner.

## Greeting Techniques
- Use warm, welcoming language
- Personalize greetings when possible
- Offer assistance proactively
- Maintain a positive, helpful tone

## Introduction Patterns
- State your name and role clearly
- Mention your key capabilities
- Invite questions or requests
- Show enthusiasm for helping
