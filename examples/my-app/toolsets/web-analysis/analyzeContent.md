# analyzeContent

Analyze the content structure and information architecture of a website.

## Overview

Understand how content is organized, what sections exist, page structure, and information hierarchy.

## Parameters

- `url` (required) - Website URL to analyze

## What This Reveals

- Main content sections
- Page hierarchy and structure
- Information organization
- Navigation patterns
- Content types and formats
- User journey flow
- Call-to-action placement
- Form structures

## Examples

### Analyze Site Structure
```
analyzeContent({ url: "https://example.com" })
```

## Use Cases

- Understanding site organization
- Learning page structure patterns
- Analyzing information hierarchy
- Studying navigation approaches
- Understanding user journey
