# fetchUrl

Fetch and retrieve the full content of a website URL.

## Overview

Get the complete HTML/content of a URL for detailed analysis or reference.

## Parameters

- `url` (required) - Website URL to fetch

## Use Cases

- Getting full page content for analysis
- Reviewing HTML structure
- Understanding code implementation
- Detailed page inspection
