# analyzeDesign

Analyze the visual design elements of a website.

## Overview

Extract design patterns, color schemes, typography, layout strategies, and visual elements from a website.

## Parameters

- `url` (required) - Website URL to analyze

## What This Reveals

- Color palettes and schemes
- Typography choices (fonts, sizes, weights)
- Layout patterns and grid systems
- Component designs and patterns
- Visual hierarchy and spacing
- Responsive design approach
- Animation and interaction patterns

## Examples

### Analyze Reference Design
```
analyzeDesign({ url: "https://example.com" })
```

## Use Cases

- User provides design inspiration URL
- Want to understand color scheme used
- Learning layout approach from reference
- Analyzing visual hierarchy
- Understanding spacing and typography choices
