# Web Analysis Tools

Tools for analyzing websites when users provide reference URLs for design inspiration or structure.

## Overview

These tools help you study existing websites to understand design patterns, content organization, and best practices. Use when users provide URLs as inspiration or examples.

## Common Use Cases

- User provides reference website for design inspiration
- Analyzing competitor site structure
- Understanding content organization patterns
- Learning from successful designs
- Finding design elements user wants to replicate
