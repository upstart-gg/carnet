# gitLog

View commit history for the project.

## Overview

Review what has changed and when. Useful for understanding project evolution, finding when features were added, or rolling back to previous states.

## Parameters

- (Optional) Various filters and options for commit history

## Examples

### View Recent Commits
```
gitLog()
```

### View Commit History
```
gitLog()
```

## Common Uses

- Understanding what changed in project
- Finding when features were added
- Locating previous implementations for reference
- Preparing for revert if needed

## Best Practices

- Use to understand project evolution
- Reference commits before reverting
- Check recent changes before starting work
