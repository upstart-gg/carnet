# Delegation Tools

Tools for the Main agent to delegate work to specialist agents (Coder, Designer, Data Architect).

## Overview

Main agent uses these tools to assign specific tasks to specialist agents who have expertise in different areas.

## Specialist Agents

### Coder
- Creates components and pages
- Implements functionality
- Manages technical architecture
- Handles TypeScript/React code

### Designer
- Creates and edits themes
- Searches for and generates images
- Makes design decisions
- Works with colors and aesthetics

### Data Architect
- Designs datasource schemas
- Plans data models
- Ensures data consistency
- Handles backend data structure

## Delegation Best Practices

### Before Delegating
1. Update memory with task context
2. Ensure dependencies exist
3. Be clear about requirements
4. Provide reference material if available

### Delegation Format
- Be specific about what's needed
- Include relevant context from memory
- Reference any related decisions
- Clarify success criteria

### After Delegation
- Wait for sub-agent completion
- Review their work
- Handle any blockers they report
- Update progress with user
