# Image Generation Tools

Tools for creating custom AI-generated images when stock photos don't match your needs.

## Overview

Generate unique, branded, or conceptual imagery using AI. Use when stock images are insufficient or you need a specific artistic vision.

## When to Generate

### Generate When:
- Need branded/unique imagery
- Specific vision stock photos can't match
- Consistent style across multiple images needed
- Conceptual or abstract imagery
- Main agent explicitly requests generation

### Search Stock Photos When:
- Need realistic photography
- Common scenes acceptable
- Quick results needed
- Cost-conscious (generation can be expensive)

## Cost Considerations

Image generation is costly - only generate when:
- Explicitly requested by main agent
- Stock images are insufficient
- Unique, branded imagery necessary
