# search_react_router_code

Search React Router code examples and source code.

## Overview

Find code examples and implementation details from React Router codebase.

## Parameters

- `query` (required) - Search term for code

## Examples

### Search for Code Example
```
search_react_router_code({ query: "useNavigate implementation" })
```

### Search for Pattern
```
search_react_router_code({ query: "data loader pattern" })
```

## Use Cases

- Finding code examples
- Understanding implementations
- Learning best practices
- Resolving implementation questions
