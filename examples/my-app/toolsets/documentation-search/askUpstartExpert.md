# askUpstartExpert

Ask the Upstart platform expert for guidance and documentation.

## Overview

Get answers about Upstart platform features, APIs, patterns, and best practices.

## Parameters

- `question` (required) - Your question about Upstart

## Examples

### Ask About Feature
```
askUpstartExpert({ question: "How do I create a custom datasource?" })
```

### Ask About Pattern
```
askUpstartExpert({ question: "What's the best way to handle form validation?" })
```

## Use Cases

- Understanding Upstart features
- Learning platform patterns
- Getting API documentation
- Resolving platform-specific issues
