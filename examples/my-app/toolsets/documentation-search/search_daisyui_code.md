# search_daisyui_code

Search DaisyUI code examples and component implementations.

## Overview

Find code examples and implementation details from DaisyUI.

## Parameters

- `query` (required) - Search term for code

## Examples

### Search for Code Example
```
search_daisyui_code({ query: "button implementation" })
```

### Search for Pattern
```
search_daisyui_code({ query: "dark mode implementation" })
```

## Use Cases

- Finding code examples
- Understanding component structure
- Learning implementation patterns
- Understanding styling approach
