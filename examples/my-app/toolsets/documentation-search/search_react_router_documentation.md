# search_react_router_documentation

Search React Router official documentation.

## Overview

Find information about React Router features, components, and patterns in official documentation.

## Parameters

- `query` (required) - Search term for documentation

## Examples

### Search for Feature
```
search_react_router_documentation({ query: "useNavigate hook" })
```

### Search for Pattern
```
search_react_router_documentation({ query: "nested routes" })
```

## Use Cases

- Learning React Router APIs
- Finding routing patterns
- Understanding hooks usage
- Getting code examples
