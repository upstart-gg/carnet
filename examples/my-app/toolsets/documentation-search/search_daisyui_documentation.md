# search_daisyui_documentation

Search DaisyUI component library documentation.

## Overview

Find DaisyUI component documentation, usage examples, and API reference.

## Parameters

- `query` (required) - Search term for documentation

## Examples

### Search for Component
```
search_daisyui_documentation({ query: "button component" })
```

### Search for Feature
```
search_daisyui_documentation({ query: "dark mode support" })
```

## Use Cases

- Finding component documentation
- Learning component API
- Finding usage examples
- Understanding theming options
