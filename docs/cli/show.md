# `carnet show`

Show detailed information about an entity

## Usage

```bash
carnet show [type] [name]
```

## Arguments

- `type`
- `name`

## Options

| Option | Description |
|--------|-------------|
| `-c, --config <path>` | path to the carnet config file |
| `-d, --dir <dir>` | content directory (default: ./carnet) |
