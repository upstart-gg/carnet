# `carnet init`

Initialize a new Carnet project

## Usage

```bash
carnet init [dir]
```

## Arguments

- `dir` (optional) - Directory where to initialize the Carnet project. Creates the directory structure with agents, skills, and toolsets folders, plus a carnet.config.json file. Defaults to `./carnet`.

## Options

| Option | Description |
|--------|-------------|
| `-c, --config <path>` | path to the carnet config file |
| `-d, --dir <dir>` | content directory (default: ./carnet) |
